人工神經網路

淺層神經網路

         是一個模仿生物神經網路的結構和功能的數學模型 ，是一種自適應系統，具備學習功能。淺層神經網路有三層 ： 輸入層、隱藏層以及輸出層 ( 如下圖 )。隱藏層和輸出層的每一個神經元經由進行線性運算和激活函數運算 ，得到此神經元的輸出。而淺層神經網路的初始權重的隨機給予的 ，此權重會因不斷的自我學習而在 [0,1] 間不停的自我修正。


激活函數

         由於神經網路只能解決線性所組成的迴歸模型 ，因此在解決非線性模型時會引入激活涵數來解決更為複雜的數據問題 。最常用的激活涵數有以下三種 :

  • Sigmoid函數 : 涵數值在 [0,1] 間，輸出的範圍也相對的穩定且容易處理，但容易造成梯度消失。      常用在生物學中。

  • Tanh函數 : 收斂速度比Sigmoid函數要快且以 0為中心的涵數，但仍有梯度消失的問題。

  • ReLu函數 : 涵數計算速度快且沒有梯度消失的問題 ，以取最大值為目的，但卻產生了dead ReLu      的問題 ，即某些神經元的死亡將導致其相對應的參數永遠無法更新 ，造成梯度在此後都只是 0。      為最常用的激活涵數之一。


    • 損失函數

             在預測中我們可以透過損失涵數中最大化或最小化之涵數 ，可以知道所建的模型是否為最佳模型。損失涵數有三個常見的指標 :

  • MAE(平均絕對值誤差) : 可以計算目標值和預測值的絕對值之和。

  • MSE(均方誤差) : 可以計算預測值和真實值之間的距離。

  • MAPE(平均絕對值誤差率) : 不僅考慮預測值與真實值的誤差， 還考慮了誤差與真實值之間的比      例。是最常被使用的目標涵數之一。


    • 優化器

             為了減低損我涵數與預測間的差距，所以我們在預測中會加入損失涵數 。常見的優化器有兩大類，分別為 GD( BGD、SGD、MBGD )和 ADAM ，在相研究中我們就選擇了用 ADMA ( AdaGrad + RMSProp )作為我們這次研究的優化器。

  • GD : 可以尋找最小的值去控制方差、更新模型,將損失涵數最小化且收斂模型。

  • ADAM : 由於GD無法進行大量的資料處理和無法投入即時更新的資料模型 ; ADAM 就為了解決此      問題而衍生出來的。
